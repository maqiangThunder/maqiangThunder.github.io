ubuntu kylin安装

发表于 2016-04-03   |   分类于 ubuntu   |     |   阅读次数

前言

本机原来是xp, 现在安装ubuntukylin-15.10-enhanced-release-amd64.iso

下载

http://www.ubuntukylin.com/

U盘启动盘

手上只有U盘, 下载一个unetbootin,官网地址在http://unetbootin.github.io/,怎么操作就不说了

安装

F12 -> U盘启动
其中可以选择自动安装,或者自己分区, 自己按需操作

输入法

这里推荐搜狗输入法,支持Ubuntu
http://www.ubuntukylin.com/application/show.php?lang=cn&id=276

flash插件

chrome安装flash插件
在ubuntu软件中心搜索安装:Pepper Flash Player－Browser plugin
之后再终端sudo update-pepperflashplugin-nonfree --install
重启chrome

没声音

不知为什么, 耳机有声音扬声器没声音,一般的方案都试过没问题
解决方案如下:
aplay -l查看设备
找到声卡设备一行,记住card number
sudo gedit /etc/asound.conf编辑

我这里是1, 里按照自己的替换
重启电脑
