十分钟学会markdown

发表于 2016-04-02   |   分类于 markdown   |     |   阅读次数

前言

最近看到简书很火, 界面简洁清新, 还有很多在git上写博客, 你就知道markdown是一种文本标记语言了, 学习下面这些标记基本能搞定大多数文章了
官网语法学习

所有的语法和文字之间都是要加上一个空格的

语法

学习基本的语法再加上多用markdown写文章就会熟练了

标题

#号都是用来表示标题的

引用

只需要在文本前加入 > 这种尖括号（大于号）即可

列表

用., +, -标记表示无序的列表, 前面可以自己加上序号(数字或者别的字母都行)

代码块

  1. 行内的
    `号
    比如: public class
  2. 整个代码块
    三个`组合
    ```
    代码在这里
    ```
    比如:

分割线

用三个或以上的*,-,_

链接

  1. 行内样式

  2. 参考样式
    [younghz的Markdown库1][数字1]
    [younghz的Markdown库2][数字2]
    [数字1]:https:://github.com/younghz/Markdown “Markdown”
    [数字2]:https:://github.com/younghz/Markdown “Markdown”

强调

在需要强调的内容两侧加上*或者_

图片

添加图片的形式和链接相似，只需在链接的基础上前方加一个！

转义

反斜杠\

windows编辑器

在线的编辑器很多, 还有比如简书就支持markdown语法
我目前使用的是Wiz的markdown插件, 因为我Wiz用的很多
