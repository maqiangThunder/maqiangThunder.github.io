windows开发-己必备的软件(持续更新)

发表于 2016-04-02   |   分类于 windows开发   |     |   阅读次数

前言

每次换一台电脑, 都要话很久来把电脑配置成自己原来的开发环境,很多软件是我一直的最爱,下面给大家分享

1.everything–文件搜索工具

如果你使用windows 自带的搜索工具就太low了, 这个工具最好是设置为开机自动启动,这样便于它建立索引,注意:windows7以以上的系统要设置使用系统管理员权限打开

2.clover–资源管理器

每天有那么几个文件夹路径要经常访问,一般我们可能做一个文件夹的快捷方式放在桌面上,这个软件一个windows资源管理器+chrome风格的标签管理器(多窗口并排显示)

3.Wiz–笔记

可以管理我们的笔记, 文章,以及帐号

4.网易云音乐–个性化的音乐播放器

歌单的形式很喜欢

5.AutoHotKey–定制windows快捷键

比如定制CMD用快捷键的方式打开

6.Cmder–CMD++

如果你觉得cmd不够强大,试试Cmder, 不过我也喜欢GitBash

7.foxmail–邮件客户端

用过很多, 这个最简洁了

8.xshell–linux的SSH客户端

自定义界面很强大

9.xftp–linux下的FTP工具

xshell系列产品

10.hbuilder–最快的HTML开发工具

11.VM–虚拟机

#windows #开发 #软件
理解Python的局部变量和全局变量
windows的一些提高效率的技巧(持续更新)
